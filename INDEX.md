## 📝 Usage Notes

### Data Binding
All ViewModels use `INotifyPropertyChanged` for automatic UI updates.

Example:
```csharp
private int _count;
public int Count
{
    get => _count;
    set => SetProperty(ref _count, value);
}
```

### Navigation
Navigation uses Frame-based routing:
```csharp
MainFrame.Navigate(new DashboardView());
```
